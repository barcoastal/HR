Key Changes

Header Headline md (24px/600) → Display lg-style (36px+/900 black). Editorial subtitle with employee count.
Search Moves to the TopBar (global search in header bar), not inline on the page.
Filters Department pills replaced by "Filters" + "Sort" buttons (surface-container-highest bg). Filter/sort logic stays the same, just different UI trigger.
Grid Uniform 3-col → Bento 4-col. First employee gets a featured 2-col-span card with skill tags + action buttons. Rest are center-aligned avatar-focused cards.
Team rows NEW: Department/team cards interspersed in the grid (3-col span). Show team icon, member count, online count, avatar stack.
Status Small colored dot → Full status label with icon (Available, In Meeting, Out of Office, Focus Mode, Traveling). Maps from employee ACTIVE/ONBOARDING/etc status.
Pagination NEW: Circular page buttons replacing infinite scroll / load-all pattern.
FAB NEW: Fixed bottom-right, gradient, person_add icon. Opens add-employee dialog.
