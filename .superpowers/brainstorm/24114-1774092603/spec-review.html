Goals

Goal 1 Visual alignment — Match the Stitch reference layouts for People Directory, Org Chart, Calendar

Goal 2 Icon migration — Replace Lucide React with Material Symbols Outlined app-wide

Goal 3 Token expansion — Add only the color tokens these pages consume

Goal 4 New UI patterns — FABs, bento grids, status labels, event chips, org tree connectors

Non-Goals

Out Week/Day calendar views (toggle renders but only Month functions)

Out Real-time presence data (statuses map from existing employee status field)

Out Calendar sync functionality (promo card is decorative)

Out Backend logic, APIs, or database schema changes

Shared Foundation

Icon Migration: Lucide → Material Symbols Outlined

Add Material Symbols font in layout.tsx. Create Icon wrapper component at src/components/ui/icon.tsx.

88 files import from lucide-react using ~90 unique icons. Complete mapping in Appendix A.

After all replacements: remove lucide-react from package.json.

Page 1: People Directory

Major Changes

Editorial header (5xl/black) · Bento 4-col grid · Featured 2-col card · Center-aligned regular cards · Status labels · Team row cards · Circular pagination · FAB

Header

"People" at text-5xl font-black. Subtitle: "Managing {count} talented individuals across {dept_count} departments." Filters + Sort buttons replace department pills.

Card Types

  • Featured card (first employee): 2-col span, horizontal layout, square avatar with grayscale→color hover, skill tags, Message + Mail buttons
  • Regular card: center-aligned, circular avatar, status label at bottom, "View Profile" ghost button
  • Team row card: 3-col span, department icon + name + member count + avatar stack + "View Team" link

Page 2: Organization Chart

Major Changes

Visual tree with CSS gradient connectors · CEO spotlight node · Director cards · Team cards with left accent · Bento stats section · FAB

Visual Org Tree

  • CEO node: centered top, gradient avatar frame, Profile + Contact buttons, shadow-xl
  • Connectors: vertical gradient lines (primary→surface-variant), horizontal solid lines
  • Director nodes: centered row below connectors, circular avatars, one highlighted with top border
  • Team cards: below each director, surface-container bg, left-4px primary border, icon + name + count
  • Depth cap: max 3 levels, deeper nodes collapse behind "View team"

Bento Stats Section

  • Org Health (2-col span): Total Employees, Retention Rate, Departments
  • Growth card: primary-container bg, white text, large percentage
  • Inter-Team card: white bg, hub icon, avatar stack

Manager Assignment

Stays as-is below the bento stats. Icons migrated, layout unchanged.
