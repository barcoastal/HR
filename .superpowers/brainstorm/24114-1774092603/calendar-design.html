Key Changes

Header Month name as editorial display text (32px/900). Subtitle shows milestone count for current week. Prev/Next/Today buttons removed from header — month navigation via arrow buttons on the calendar itself.
View toggle NEW: Month/Week/Day pill toggle (surface-container-low bg, active tab in white with shadow). Month view is default. Week/Day views are future — Month only for now, but toggle renders for visual consistency.
Calendar container Grid wrapped in surface-container-low rounded-[2rem] container. Day labels use outline color, uppercase, 2px letter-spacing, 900 weight.
Day cells Small flat cells → Large rounded-2xl cells (min-height 120px). White bg (surface-container-lowest). Previous/next month days at 40% opacity.
Events in cells Colored dots → Inline event chips with text labels. Color-coded by event type: primary/10 for meetings, tertiary-container/10 for birthdays, tertiary-fixed for anniversaries, error-container/20 for holidays.
Today highlight primary-fixed bg, 2px primary/20 border, "Today" pill badge, bold day number in primary color.
Legend + detail panel REMOVED: Legend bar and click-to-select detail panel. Events are now self-describing inline. Clicking a day could show a popover instead (future enhancement).
Summary cards 4 flat summary cards → REPLACED by glass "Upcoming Highlights" panel (mini calendar date icons + event details) and dark inverse-surface promo card.
Insights panel NEW: Glassmorphism card with upcoming highlights. Mini date icons (month header + day number). Shows next 2-3 notable events.
Promo card NEW: Dark inverse-surface card with "Pro Tip" badge, CTA button, decorative gradient + oversized icon watermark. Promotes calendar sync feature.
FAB NEW: Fixed bottom-right, primary bg, event icon.
