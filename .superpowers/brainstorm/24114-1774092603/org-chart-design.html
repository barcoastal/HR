Key Changes

Header Same editorial treatment — extrabold title, descriptive subtitle. "All Departments" filter + "Export Chart" primary button.
Tree visualization Indented list → Visual tree with CSS gradient connector lines (vertical: primary→surface-variant gradient, horizontal: solid surface-variant). Cards centered at each level.
CEO node Spotlight treatment: gradient-bordered avatar frame, shadow-xl, Profile + Contact buttons. Larger than director nodes.
Director nodes Photo + name + title. Circular avatar. Ghost border. One gets a top-4px primary accent border (highlights key role).
Team cards Below each director: surface-container cards with left-4px primary border, Material icon, team name, member count. Replace current department grid.
Stats section Flat 4-col stat cards → Bento grid below the tree. 2-col "Org Health" card with 3 metrics, primary-container "Growth" card, white "Inter-Team" card with avatar stack.
Manager Assignment Stays as a separate section/tab — not shown in Stitch but keep as-is since it's functional admin tooling.
FAB NEW: Fixed bottom-right, primary bg, add_moderator icon.

Data Mapping

CEO node Employee with no manager (root of tree). Currently identified by buildTree() in org-tree.tsx.
Director nodes Direct reports of the root employee. Depth 1 in current tree.
Team cards Departments, grouped under their head's director node. Uses existing department data (name + employee count).
Org Health metrics Total employees (count), Retention rate (calculated from offboarded/total), Departments (count). Growth % is static/decorative for now.
Inter-Team card Decorative — avatar stack of recent cross-department collaborators. Static for now.
